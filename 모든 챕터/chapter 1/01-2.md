이클립스 개발 구축
이클립스는 무료로 사용할 수 있는 오픈소스 통합 개발 환경(IDE:Integrated Development Environment)이다.
IDE란 프로젝트 생성, 자동 코드 완성, 디버깅 등과 같이 개발에 필요한 여러 가지 기능을 통합적으로 제공하는 툴이다.
이클립스는 기본적으로 자바 프로그램을  개발 할 수 있도록  구성되어있지만, 개발자가 추가적으로 플로그인(plugin)을 설치하면 웹 애플리케이션 개발,C,C++애플리케이션 개발등 다양한 개발 환경을 구축할수 있다.
퍼스펙티브(Perspecive)는 이클립스에서 프로젝트를 개발할때 유용하게 사용할 수있는 뷰(View)들을 미리 묶어서 이름을 붙여놓은것
Eclipse IDE for ENterprise Java and Web Developers 버전의 이클립스는 기본적으로 Java EE 퍼스펙티브를 보여줍니다.
Java 퍼스펙티브에서 제일 많이 사용되는 뷰는 Package Explorer뷰 입니다.
Package Explorer뷰에서 프로젝트를 관리하고, 자바 소스 파일을 생성 및 삭제하는 작업을 합니다.
Console 뷰에서는 프로그램에서 출력하는 내용을 볼 수 있습니다.
